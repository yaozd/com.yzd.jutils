## js、java的base64相互加密、解密,用于前后台数据传输

####  适用场景：
```
1、js、ajax前端传参到后端，中文乱码问题；
2、js、ajax前端json传参到后端，json参数含有特殊字符报错问题；
提醒：所以提前用base64处理json字符串或参数

```
