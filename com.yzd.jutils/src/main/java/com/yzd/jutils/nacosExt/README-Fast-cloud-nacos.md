# fast-cloud-nacos
- [https://github.com/fafeidou/fast-cloud-nacos](https://github.com/fafeidou/fast-cloud-nacos)
- [nacos注册grpc并实现负载均衡调用](https://blog.csdn.net/qq_37362891/article/details/103920438)

## 前言

> `fast-cloud-nacos`项目致力于打造一个基于nacos为注册中心，结合企业开发习惯，总结的一些基本的实现方式。

### 组织结构

``` ssh

|---------------fast-boot-examples                   springboot原理的介绍
```
