## 考察方面
1. 参与过大量用户，高并发项目，有过相关实践
2. 对某一技术方向有较深的研究
3. 基础掌握的扎实
4. 解决问题能力强，遇到并处理过有挑战性的问题
5. 学习能力，逻辑能力

1. 有经验：关注经验落地及思考
2. 无经验：关注基础，还有潜力项（思考设计能力，解决问题能力，学习沟通能力）
